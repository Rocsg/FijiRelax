Package organization:

* curvefit contains the optimizers and the associated routines for curve fitting T1/T2/PD params using samples of MRI data

* gui contains classes describing the graphical user interface of the Fiji pluginged	

* lma contains linear matrix algebra functions used for the curve fitting algorithms

* mrialgo contains the data structure description (HyperMap), RiceEstimator functions, and utils for MR data handling
